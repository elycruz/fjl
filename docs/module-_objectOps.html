Creates a value `of` given type; Checks for one of the following construction strategies (in order listed): ``` // - If exists `(value).constructor.of` uses this. // - If value is of one String, Boolean, Symbol, or Number types calls it's // constructor as a function (in cast form; E.g., `constructor(...args)` ) // - Else if constructor is a function, thus far, then calls constructor using // the `new` keyword (with any passed in args). ```
